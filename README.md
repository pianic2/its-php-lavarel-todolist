## 🚀 Overview
What started as a coding exercise evolved into a complete, structured management tool. While the concept—a Todo List—is simple, the real goal was to build a professional-grade application focused on clean code, scalability, and maintainability.
------------------------------
## ⚙️ Key Features

* List Management: Create and organize multiple task lists.
* Task Control: Add, edit, and mark tasks as complete.
* Smart Filtering: Filter tasks by status for better focus.
* Collaboration: Share lists with other users in real-time.
* Email Notifications: Receive automated alerts when invited to a list.
* Server-Side Rendering: Built with Laravel Blade for a fast, SEO-friendly interface.

------------------------------
## 🧠 What I Learned
This project allowed me to implement professional development workflows and solve concrete architectural challenges:

* MVC Architecture: Structuring a clean backend using Laravel’s core patterns.
* Data Modeling: Managing complex relationships (One-to-Many and Many-to-Many).
* Authentication & Authorization: Securing user data and managing access control.
* RESTful Routing: Designing clear, consistent, and predictable URL structures.
* Validation & UX: Handling data integrity and providing meaningful user feedback.
* Integrated Logic: Going beyond basic CRUD by implementing notification systems and business rules.

------------------------------
## 🛠️ Tech Stack

* Framework: Laravel (PHP)
* Database: MySQL (via Docker)
* Frontend: Blade Templates & Vite
* Testing: PHPUnit
* DevOps: Docker Compose & GitHub Actions (CI)
